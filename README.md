# Cuadro de Mandos de Venta de Vehículos

Este proyecto es una aplicación web interactiva desarrollada en Python utilizando **Streamlit** y **Plotly Express**. La aplicación web proporciona un análisis exploratorio de datos (EDA) para un conjunto de datos sobre anuncios de venta de coches en EE. UU.

### Funcionalidades:
- **Histograma interactivo:** Permite visualizar la distribución del kilometraje (`odometer`) de los vehículos usando casillas de verificación.
- **Gráfico de dispersión interactivo:** Permite analizar la relación entre el kilometraje (`odometer`) y el precio (`price`) de los vehículos.

### Cómo ejecutar la aplicación localmente:
1. Asegúrate de tener el entorno virtual activo.
2. Ejecuta el comando: `streamlit run app.py`
